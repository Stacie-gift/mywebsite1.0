# Stacie's Personal Website

A beautiful, responsive personal website built with HTML, CSS, and JavaScript featuring an underwater theme with stunning animations and effects.

## Features

- 🌊 **Underwater Theme**: Beautiful gradient backgrounds with floating jellyfish and bubble animations
- 📱 **Fully Responsive**: Works perfectly on desktop, tablet, and mobile devices
- ✨ **Smooth Animations**: CSS animations and JavaScript interactions for an engaging experience
- 🎨 **Glassmorphism Design**: Modern glass-like effects throughout the interface
- 📝 **Contact Form**: Functional contact form with validation and notifications
- 🚀 **Performance Optimized**: Lightweight and fast loading

## Sections

1. **About Me**: Personal introduction with skills and social links
2. **Projects**: Showcase of development projects with technologies used
3. **Blog**: Latest blog posts and articles
4. **Memories**: Special moments and experiences
5. **Community**: Involvement in tech community activities
6. **Contact**: Contact information and form

## File Structure

```
MyWebsite/

├── styles.css          # All CSS styling and animations

└── README.md           # This file
```

## How to Use

1. **Open the website**: Simply open `index.html` in any modern web browser
2. **Navigate**: Use the navigation menu to jump to different sections
3. **Mobile menu**: On mobile devices, tap the hamburger menu to access navigation
4. **Contact form**: Fill out the contact form to send a message (currently simulates submission)

## Customization

### Colors
The website uses a purple-blue gradient theme. You can modify colors in `styles.css`:

```css
/* Main gradient colors */
background: linear-gradient(135deg, #0f172a 0%, #1e1b4b 25%, #1e3a8a 50%, #3730a3 75%, #0f172a 100%);

/* Accent colors */
--primary: #a855f7;
--secondary: #3b82f6;
```

### Content
- **Personal Information**: Update the name, title, and description in `index.html`
- **Skills**: Modify the skills list in the About section
- **Projects**: Add your own projects in the Projects section
- **Blog Posts**: Update the blog content with your articles
- **Contact Info**: Update email, location, and social media links

### Animations
The website includes several animations:
- Floating jellyfish and bubbles

- Text shimmer effects
- Smooth scrolling
